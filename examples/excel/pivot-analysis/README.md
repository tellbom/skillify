# excel/pivot-analysis

A prompt-only Claude Agent Skill (no scripts/dependencies) that teaches an agent how to
turn flat tabular data into a pivot-table-style grouped summary.

## Install

```
skillctl install excel/pivot-analysis
```

## Try it

Paste a small CSV into the conversation and ask the agent to "pivot this by region, summing
amount" — `SKILL.md` gives it the steps to follow.
